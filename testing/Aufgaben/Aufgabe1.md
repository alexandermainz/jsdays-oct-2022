# Aufgabe 1

Mapping testen

```
{
  id: 1,
  firstName: 'Klaus',
  lastName: 'Müllermeier',
}
```

```
{
  id: 1,
  first_name: 'Klaus',
  last_name: 'Müllermeier',
  username: 'kmueller'
}
```
